# Beauty Scanner App

##  Project Overview
The Beauty Scanner App is an image classification system designed to identify different beauty and cosmetic products using image recognition technology. This project applies machine learning techniques to classify images into predefined cosmetic categories.

---

##  Image Classification Types
The system classifies images into the following ten (10) categories:

1. Lipstick  
2. Mascara  
3. Eyelash Curler  
4. Compact Powder  
5. Eyeshadow Palette  
6. Foundation Bottle  
7. Makeup Brush  
8. Eyeliner Pen  
9. Makeup Sponge  
10. False Eyelashes
